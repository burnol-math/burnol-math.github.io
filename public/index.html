Jean-François Burnol GitLab Projects

Irwin

This one-page site serves as a brief mathematical introduction to the formulas discovered by the author (and published in 2024) regarding Kempner-Irwin harmonic series. It contains links to the repository where one finds SageMath scripts implementing them, and also provides links to the research papers.

Kempner

Legacy Maple code (in its 2024 version with English comments) as used by the author some years ago to implement the formulas he had obtained relative to the Kempner series. Almost superseded by the SageMath scripts from Irwin, except that the Maple code is the only implementation so far in the case of excluding multiple digits at once.
